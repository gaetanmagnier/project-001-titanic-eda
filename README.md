# Project 001 — Titanic Exploratory Data Analysis

## Project Overview
Portfolio Project 001 — This project explores the Titanic passenger dataset to identify the factors most strongly associated with survival.
The analysis focuses on understanding the data through visualization, descriptive statistics and exploratory data analysis rather than maximizing predictive accuracy.

## Objective
Determine which passenger characteristics appear to have the strongest relationship with survival during the Titanic disaster.

## Dataset
Source:
Kaggle Titanic Dataset

## Research Questions
- Does passenger class influence survival?
- Did women survive more frequently than men?
- Did age affect survival?
- Was ticket price associated with survival?
- Did travelling with family matter?
- Were some embarkation ports associated with higher survival rates?

1. Load the dataset

2. Inspect variables

3. Explore missing values

4. Visualize important relationships

5. Draw conclusions supported by the data
